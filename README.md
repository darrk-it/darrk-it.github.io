## Deployment

This website is hosted using **GitHub Pages**. To view the live site:

1. Go to the repository settings.
2. Enable **GitHub Pages** from the main branch (root folder).
3. The site will be available at `https://darrk-it.github.io/`.

## Usage

- **Invite the Bot:** Use the invite link on the homepage to add Promptly to your Discord server.
- **View Policies:** Navigate to the **Terms of Service** and **Privacy Policy** pages for detailed information on usage and data handling.
